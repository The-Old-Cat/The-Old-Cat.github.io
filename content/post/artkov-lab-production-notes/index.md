# Добро пожаловать

Это моя рабочая база знаний. Здесь собраны настройки, чек-листы, примеры конфигураций и решения типовых задач, которые встречаются в повседневном администрировании.

Никакой теории — только то, что было поднято, настроено и проверено в работе.

---

### Что здесь собрано

Мой стек — это классический набор инженера инфраструктуры:

| Область | Что внутри |
| :--- | :--- |
| **Виртуализация** | Hyper-V, VMware ESXi, Proxmox (установка, сеть, LVM, RAID, проброс устройств, бэкапы) |
| **Операционные системы** | Linux (Ubuntu/Debian), Windows Server (AD, GPO, PowerShell) |
| **Сети** | MikroTik (маршрутизация, VPN, VLAN), OpenVPN, SSH, фаерволы |
| **Мониторинг** | Zabbix, NetData, Wazuh SIEM |
| **Автоматизация** | Python, PowerShell, Bash-скрипты, Git |

---

### Как устроены заметки

Всё основано на Obsidian. Заметки связаны между собой — если видите ссылку на другую тему, переходите, там детали.

Основные типы материалов:
- **Инструкции** — пошаговые руководства по установке и настройке.
- **Чек-листы** — краткие списки действий для быстрого развёртывания.
- **Cheat-sheets** — шпаргалки по командам и синтаксису.
- **Решения проблем** — описание ошибок и способов их исправления.

---

### Кому это может быть полезно

- Системным администраторам, которые работают с похожим стеком.
- Всем, кто использует системы виртуализации.
- Тем, кто просто ищет компактные инструкции без воды.

---

### Как это использовать

Проще всего — искать через поиск. Если нужна тема — введите ключевое слово.

Начинать можно с любого раздела. Материалы не выстроены в линейную последовательность, каждый сам решает, что ему нужно.

Если нашли ошибку или знаете лучшее решение — пишите, буду благодарен за обратную связь.
