Привет, меня зовут Артём.

Я системный администратор с опытом построения надёжной инфраструктуры для среднего бизнеса. Занимаюсь автоматизацией, виртуализацией и всем, что связано с серверами, сетями и отказоустойчивостью.

### Мой технологический стек

В своей работе использую широкий набор инструментов, которые подбираю под конкретную задачу:

- **Виртуализация:** Proxmox VE, VMware ESXi, Hyper-V
- **ОС и сервисы:** Windows Server (AD, GPO), Linux (Ubuntu/Debian)
- **Сети:** MikroTik, VLAN, VPN (OpenVPN, WireGuard)
- **Мониторинг и безопасность:** Zabbix, Netdata, Wazuh SIEM
- **Резервное копирование:** Veeam, Acronis (стратегия 3-2-1)
- **1С:** Администрирование серверов (Linux/Windows), RAC, MSSQL, PostgreSQL
- **Автоматизация:** PowerShell, Bash, Python

### Из чего состоит мой подход

1. **Документирование всего.** Любое решение фиксирую в заметках — это экономит время в будущем.
2. **Автоматизация рутины.** Если задача повторяется — она должна выполняться скриптом.
3. **Надёжность прежде всего.** Инфраструктура должна быть предсказуемой и восстанавливаемой.
